## Dependencies

### EXTERNAL LIBRARIES
- Next.js: 16.0.4
- React: 19.2.0
- Tailwind CSS: 4.1.17
- Styled JSX: 5.1.6

## Build Instructions
1. Install dependencies
    ```bash
    npm install
    ```
2. Run development server
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. Build for production
    ```bash
    npm run build
    npm start
    ```
    This compiles the app for production and runs the production server on [http://localhost:3000](http://localhost:3000).
